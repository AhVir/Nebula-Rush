# Nebula Rush 🔫☄️  

**[PLAY NOW](https://ahvir.github.io/Nebula-Rush/)** *(It’s just point + shoot!)*  

A minimalist mouse-shooter where you blast asteroids before they crush you. No upgrades. No tutorials. Just **survival**.

### **Controls**  
- **Aim** → Mouse  
- **Shoot** → Left Click  
- **Goal** → Don’t die.  

### **Tech Stack**  
- Vanilla JS + Canvas  
- Hosted on GitHub Pages  
- 0 dependencies
